# Prompt Imagen Pantalla De Carrera

## Prompt Base

Concept art / high fidelity UI mockup of an Android augmented reality slot car racing game in landscape mode, race screen only, camera view pointed at a real wooden table with a printed black and white AR marker, a miniature dual-lane AR racetrack anchored above the marker, two tiny slot cars racing on the track, player one red and player two green, friendly miniature social arcade feeling, Toy Track Signal visual direction, compact perimeter HUD, dark semi-transparent overlay panels, high contrast UI, amber primary accents for countdown and main action, bright cyan status chips for tracking and ready state, minimal text, lap counter visible, clear position and race status, one large dominant acceleration button in the lower corner, polished Unity mobile game interface, believable AR compositing, clean lighting, readable over live camera, fast and exciting but not cluttered, the track remains the main visual focus, academic prototype that looks functional and playable.

## Prompt Afinado

Landscape Android game screenshot, AR slot car racing game, race tab, real tabletop camera background, single printed marker on table, shared miniature racetrack floating stably above the table, two parallel lanes, red car vs green car, short race in progress, Toy Track Signal art direction, playful but clean, mini toy-like scale, compact HUD around the edges, dark translucent top bar with network and tracking status, lap counter and player identity, amber countdown/action highlight, cyan confirmation chips, bold readable typography, very little text, one big acceleration pad, camera-first layout, center area kept clear for the AR track, high contrast, polished UI mockup, modern mobile game UX, realistic AR placement, no extra menus, no inventory, no steering wheel, no joystick.

## Negative Prompt

No futuristic spaceship cockpit, no first person driving view, no crowded dashboard, no purple theme, no generic sci-fi hologram overload, no extra buttons, no complex telemetry wall, no cartoon explosion effects, no dark unreadable UI, no portrait orientation, no split screen, no console controller prompts, no open world city background.

## Parametros Sugeridos

- Aspect ratio: 16:9
- Camera: slightly top-down handheld phone view
- Style: polished mobile game UI mockup
- Focus: race screen only
